# Gwent Forge

### Unofficial Fan-Made Deck Builder

Build your decks before game night.

**Less Setup, More Gwent.**

---

Gwent Forge is a fan-made web application that helps players create, manage, and organize decks for **Gwent: The Legendary Card Game**.

Instead of spending valuable game night time building decks, players can prepare them in advance, save multiple deck configurations, and arrive ready to play.

## Features

- Create and manage Gwent decks
- Browse cards, factions, and abilities
- Save multiple deck configurations
- Support for multiple languages
- Fast deck preparation before game night

## Tech Stack

- React
- Vite
- Tailwind CSS
- Motion for React
- tsParticles
